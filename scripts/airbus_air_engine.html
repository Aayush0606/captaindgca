#1. The Full Authority Digital Engine Control (FADEC) controls which of the following functions:

FADEC System Functions

1. Control of Gas Generator

  • Fuel flow control

  • Acceleration and deceleration schedules

  • Variable bleed valve and variable stator vane schedules

  • Turbine clearance control

  • Idle setting

2. Protection Against Engine Exceeding Limits

  • N1 and N2 overspeed protection

  • Monitoring of EGT during engine start

3. Power Management

  • Automatic control of engine thrust rating

  • Computation of thrust parameter limits

  • Manual power management based on thrust lever position

  • Automatic power management (A/THR demand)

4. Automatic Engine Starting Sequence

  • Control of:

    • Start valve (ON/OFF)

    • HP fuel valve

    • Fuel flow

    • Ignition (ON/OFF)

  • Monitoring of:

    • N1, N2, Fuel Flow (FF), Exhaust Gas Temperature (EGT)

  • Initiation of abort and recycle (on the ground only)

5. Manual Engine Starting Sequence

  • Passive monitoring of engine

  • Control of:

    • Start valve

    • HP fuel valve

    • Ignition

6. Thrust Reverser Control

  • Actuation of blocker doors

  • Engine setting during reverser operation

7. Fuel Recirculation Control

  • Fuel recirculation to tanks based on:

    • Engine oil temperature

    • Fuel system configuration

    • Flight phase

8. Transmission of Engine Parameters & Monitoring Information to Cockpit Indicators

  • Primary engine parameters

  • Starting system status

  • Thrust reverser system status

  • FADEC system status

9. Detection, Isolation, and Recording of Failures

10. FADEC Cooling

#2. The FADEC is normally powered by:

Each powerplant has a FADEC (Full Authority Digital Engine Control) system.
FADEC is a digital control system that performs complete engine management.
FADEC has two-channel redundancy, with one channel active and one in standby.
If one channel fails, the other automatically takes control.
The system has a magnetic alternator for an internal power source.
FADEC is mounted on the fan case.
The Engine Interface Unit (EIU) transmits to FADEC the data it uses for engine management
